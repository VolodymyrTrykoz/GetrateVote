Что мне нужно сделать для участия в конкурсе? // Що я маю зробити для участі у конкурсі?

  • 1. Быть старше 18 лет; // 1. Бути старшим 18 років;
  • 2. Авторизоваться в Facebook; // 2. Авторизуватися у Facebook;
  • 3. Выбрать дату для предсказания и написать свои предсказания курса продажи доллара в отведенных для этого полях; // 3. Обрати дату для передбачення і написати своє передбачення курсу продажу долара у відповідному полі;
  • 4. Сделать репост прогноза на свою стенку в Facebook с: // 4. Зробити шер (репост) прогнозу у свій життєпис Facebook, в якому мають бути:
    • a. пометкой, что это могут видеть все; // а. відмітка, що цей шер можуть бачити усі користувачі Facebook;
    • b. хэштегом #курснавыигрыш; // b. хештег #курснавиграш;
    • c. отметкой одного друга в твоем шере; // c. відміткою одного друга у твоєму шері;

В какой период проводятся розыгрыши призов? // У який період проводяться розіграші подарунків?

В рабочие дни с 10 октября 2018 по 31 октября 2018. // У робочі дні з 10 по 31 жовтня 2018 року.

Могу ли я принимать участие в конкурсе более одного раза? // Чи можу я брати участь у конкурсі більше одного разу?

Да, но не более одного предсказания за дату. Кроме того, каждое предсказание будет означать новый репост на твою страницу в Facebook и отмечание нового друга. // Так, але не більш ніж одне передбачення на дату. Крім того, кожне передбачення буде означати новий шер на твою сторінку у Facebook та відмітку одного нового друга.

По какому принципу определяются победители? // За яким принципом визначаються переможці?

Ежедневно победителем становится человек, который угадал средний по Киеву курс продажи доллара на 12:00 этого дня. Он будет высчитываться по данным всех обменников Киева, находящихся в базе сайта dekurs.com.ua // Щодня переможцем стає людина, що вгадала середній за Києвом курс продажу долара на 12:00 цього дня. Середній курс буде вираховуватися за даними усіх пунктів обміну валют міста Києва, що знаходяться у базі даних сайту dekurs.com.ua

А что, если я не один правильно угадаю курс? // А якщо я не один вгадаю курс?

Тогда Победителем среди двух (или большего количества) угадавших станет тот, кто наберет больше лайков своим шером в Facebook. // Тоді Переможцем серед двох (або більшої кількості) учасників, що вгадали, стане той, хто набере більшу кількість лайків своїм шером у Facebook.

Как я узнаю, что выиграл Приз? // Як я дізнаюся, що виграв Подарунок?

Твой профиль появится на странице “Победители” сайта конкурса. Более того, ты получишь оповещение на свой профиль в Facebook! // Твій профіль з'явиться на сторінці ”Переможці” сайту конкурсу. Більш того, ти отримаєш сповіщення у свій профіль на Facebook!

Как мне получить Приз, если я победитель? // Як мені отримати Подарунок, якщо я переможець?

Условия получения Приза обсуждаются дополнительно через твою страницу в Facebook. // Умови отримання Подарунку обговорюються додатково через твою сторінку у Facebook.
